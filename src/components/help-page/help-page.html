CoReBIN Comparison and Evaluation of metagenomic binning

Help

CoReBIN is a user-friendly web-based tool to compare the results of different binning methods and to aid manual refinement of the bins. The overall binning result of two different methods can be visually compared in a chord diagram. Individual bins can be inspected and refined using GC%, coverage and tetranucleotide frequencies.

Input data

  • Metagenomic assembly (fasta format)
  • Coverage table (see format details below) not required
  • One or more binning results (see format details below)

Getting started

  1. Upload your metagenomic assembly (fasta format). It is beneficial to also upload a table with coverage information on your assembly.
  2. Next you can upload the result of your binning in the "Overview" tab. To do so, you need a file in which the contig ID if followed by the bin number (e.g. contig binID).
